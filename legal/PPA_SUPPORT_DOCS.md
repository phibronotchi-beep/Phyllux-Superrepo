## Filing Checklist

**Before Filing:**
- [ ] Verify all 25 figures are final
- [ ] Map figures to claims
- [ ] Confirm code commits timestamped
- [ ] Review qualified language
- [ ] Confirm method spine in all technologies

**During Filing:**
- [ ] Include PPA_EVIDENCE_MAP.md
- [ ] Reference GitHub repos and commit hashes
- [ ] Cite figures by ID
- [ ] Link embodiments to fusions
- [ ] Include URLs to public repos

**After Filing:**
- [ ] Update PRIOR_ART_REGISTRY.md
- [ ] Add PPA filing date
- [ ] Mark briefs as ready to send
- [ ] Update CONTROL_PANEL.md
