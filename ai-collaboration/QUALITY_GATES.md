# Quality Gates

**Purpose:** Quality checks to ensure all outputs meet standards before use.

**Last Updated:** January 25, 2026

---

## Quality Checks

### Qualified Language Check
- [ ] All performance claims use qualified language
- [ ] No absolute claims ("guarantees", "proven", "definitely")
- [ ] Maturity framework applied

### Technical Accuracy Check
- [ ] Technical details accurate
- [ ] Code references correct
- [ ] Figure references valid
- [ ] Prior art dates correct

### Professional Tone Check
- [ ] Professional and respectful
- [ ] Clear and concise
- [ ] Appropriate for audience
- [ ] Mission alignment emphasized (if Tier 2)
