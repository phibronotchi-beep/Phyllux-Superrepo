# Quick Start Guide

**Purpose:** 5-minute orientation to the Phyllux Superrepo

## If You Only Have 5 Minutes

1. **Supporting PPA Filing?**  
   → Go to [PPA_EVIDENCE_MAP.md](strategy/PPA_EVIDENCE_MAP.md)

2. **Need to Send a Partner Brief?**  
   → Check [partnerships/briefs/](partnerships/briefs/) for Tier 1 urgent briefs  
   → Follow [FAST_TRACK_WORKFLOW.md](partnerships/FAST_TRACK_WORKFLOW.md) to generate new briefs

3. **Want to See Top Opportunities?**  
   → Read [TOP_5_LANES.md](strategy/TOP_5_LANES.md)

4. **Daily Operations?**  
   → Use [CONTROL_PANEL.md](CONTROL_PANEL.md)

## If You Have 30 Minutes

1. Read [README.md](README.md) - Understand what this repo is and isn't
2. Review [SYSTEM_CONTEXT.md](SYSTEM_CONTEXT.md) - How the three existing repos relate
3. Scan [TOP_5_LANES.md](strategy/TOP_5_LANES.md) - Your highest-value commercial opportunities
4. Check [partnerships/briefs/_URGENT_neuralink_mesh.md](partnerships/briefs/_URGENT_neuralink_mesh.md) - Tier 1 BCI brief (available on request)
5. Read [HANDOFF_GUIDE.md](HANDOFF_GUIDE.md) - Critical if delegating during treatment

## If You Have 2 Hours

Complete the 30-minute orientation, then:

6. Read [PPA_EVIDENCE_MAP.md](strategy/PPA_EVIDENCE_MAP.md) - Understand cross-repo evidence
7. Review all four tier templates in [partnerships/briefs/](partnerships/briefs/)
8. Read [REVENUE_MODEL.md](strategy/REVENUE_MODEL.md) - Understand deal economics
9. Review [CURSOR_RULES.md](ai-collaboration/CURSOR_RULES.md) - AI safety guardrails
10. Plan your first week using [CONTROL_PANEL.md](CONTROL_PANEL.md)
