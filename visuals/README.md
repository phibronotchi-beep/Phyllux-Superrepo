## Image Generation

All images are generated using the FAL API with high-resolution settings. The generation scripts are excluded from git for security (API keys stored in environment variables only).

**Note:** All PNG images are generated without text to avoid FAL misspelling issues. SVG files may contain text as they are vector graphics.

## Usage

Images are embedded throughout the documentation using standard markdown image syntax:
```markdown
![Image Description](path/to/image.svg)
```

## File Formats

- **SVG**: Vector graphics for all diagrams and visualizations (scalable, can include text)
- **Note**: All PNG images have been replaced with comprehensive SVG files for better quality and scalability
